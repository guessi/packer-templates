### Overview

Packer templates for creating **version control** images


### Prerequisites

* Packer (>= 0.10.2)
* QEMU (>=2.0.0)
* libguestfs (>=1.24.0)


### Usage

    $ sudo ./setup.sh
    $ sudo /opt/packer build <json>


### Support OS distributions

* Ubuntu Server 12.04 LTS
* Ubuntu Server 14.04 LTS
* Ubuntu Server 16.04 LTS


### Support output format

* ova
* qcow2
